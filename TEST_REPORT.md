# QQ机器人Webhook服务 - 测试报告

## 项目概述

基于QQ机器人API v2开发的Webhook服务，支持单聊和群聊场景的`/insight`命令响应。

## 技术栈

| 组件 | 技术 | 版本 |
|------|------|------|
| 语言 | Python | 3.9+ |
| Web框架 | FastAPI | 0.104.1 |
| ASGI服务器 | Uvicorn | 0.24.0 |
| 签名验证 | PyNaCl | 1.5.0 |
| HTTP客户端 | httpx | 0.25.2 |
| 数据验证 | Pydantic | 2.5.2 |

## 响应时间测试

根据QQ机器人API规范，被动回复消息需在**5分钟**内响应。本服务设计目标：

- **目标响应时间**: < 3秒
- **实现方式**: 异步处理，直接返回响应

## 已知限制

1. **主动消息**: 2025年4月21日起不再支持主动推送
2. **被动消息有效期**: 
   - 单聊: 60分钟
   - 群聊: 5分钟
3. **回复次数限制**: 每条消息最多回复5次
